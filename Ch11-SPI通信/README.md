# 第11章 SPI通信

## 11-1 SPI通信协议

### SPI 通信

- **SPI（Serial Peripheral Interface）** 串行外设接口
- 高速、全双工、同步的串行通信总线
- 四根通信线：
  - **SCK（Serial Clock）**：串行时钟线
  - **MOSI（Master Output Slave Input）**：主机输出/从机输入
  - **MISO（Master Input Slave Output）**：主机输入/从机输出
  - **SS（Slave Select）**：从机选择（片选）

### 引脚配置

- **输出引脚**：配置为**推挽输出**
- **输入引脚**：配置为**浮空输入**或**上拉输入**

### 移位示意图
    SPI主机                              SPI从机
```c
    ┌─────────────┐                     ┌─────────────┐
│  ┌────────┐ │      MOSI ────────▶ │  ┌────────┐ │
│  │10100110│◀──┐                   │  │01010101│◀─┘
│  └────────┘   │      MISO ◀───────│  └────────┘
│       ▲       │                   │       ▲
│  波特率发生器  │                   │       │
└─────────────┘                     └─────────────┘
SCK ──────────────────────────▶
```
- 主机和从机各有一个 **8位移位寄存器**
- 主机移位寄存器数据从 **MOSI** 移出 → 从机移位寄存器
- 从机移位寄存器数据从 **MISO** 移出 → 主机移位寄存器
- 在 **SCK** 时钟驱动下，两个寄存器同时移位，完成数据交换

### SPI 时序基本单元

#### 起始条件与终止条件

| 条件 | 说明 | 波形 |
|------|------|------|
| **起始条件** | SS 从高电平切换到低电平 | `SS: ─┐ └────` |
| **终止条件** | SS 从低电平切换到高电平 | `SS: ───┐ ┌──` |

#### 交换一个字节（模式1）【了解】

- **CPOL（时钟极性）= 0**：空闲状态时，SCK 为低电平
- **CPHA（时钟相位）= 1**：SCK 第一个边沿**移出**数据，第二个边沿**移入**数据
SS:   ───┐ └──────────────────────────────┐ ┌──
SCK:  ───┘    ┌──┐    ┌──┐    ┌──┐   ...  └──
└──┘    └──┘    └──┘
MOSI: ──────┐  B7  ┌──┐ B6 ┌──┐ B5  ...
└─────┘  └────┘  └────
MISO: ──────┐  B7  ┌──┐ B6 ┌──┐ B5  ...
└─────┘  └────┘  └────

#### 交换一个字节（模式0）【掌握】

- **CPOL（时钟极性）= 0**：空闲状态时，SCK 为低电平
- **CPHA（时钟相位）= 0**：SCK 第一个边沿**移入**数据，第二个边沿**移出**数据
SS:   ───┐ └──────────────────────────────┐ ┌──
SCK:  ───┐    ┌──┐    ┌──┐    ┌──┐   ...  ┌──
└──┐  └──┘    └──┘    └──┘      └──┘
MOSI: ──────┤  B7  ┌──┐ B6 ┌──┐ B5  ...
└─────┘  └────┘  └────
MISO: ──────┤  B7  ┌──┐ B6 ┌──┐ B5  ...
└─────┘  └────┘  └────

### SPI 四种模式

| 模式 | CPOL | CPHA | 空闲时SCK | 采样边沿 |
|------|------|------|-----------|----------|
| 模式0 | 0 | 0 | 低电平 | 上升沿移入，下降沿移出 |
| 模式1 | 0 | 1 | 低电平 | 下降沿移出，上升沿移入 |
| 模式2 | 1 | 0 | 高电平 | 下降沿移入，上升沿移出 |
| 模式3 | 1 | 1 | 高电平 | 上升沿移出，下降沿移入 |
