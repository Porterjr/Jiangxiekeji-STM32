## 3-1 GPIO输出

### GPIO 基本结构

APB2 总线 → GPIOA (PA0~PA15)  
→ GPIOB (PB0~PB15)

**寄存器**：一段特殊的存储器，内核可通过 APB2 总线对寄存器进行读写，从而完成输出电平和读取电平的功能
- **输出寄存器**：写 `1` → 对应引脚输出高电平；写 `0` → 低电平
- **输入寄存器**：读取为 `1` → 对应端口目前是高电平；读取为 `0` → 低电平
- **驱动器**：用来增加信号的驱动能力

### I/O 端口位基本结构
VDD(3.3V) ── 上拉电阻 ──┐
│
├── 保护二极管
│
├── 输入驱动器
│
├── 输出控制 → PMOS / NMOS → VDD / VSS(0V)
│
├── 输出数据寄存器
│
└── 复用功能输出

- 来自片上外设（模拟输入、复用功能输入）
- 输出数据寄存器 / 复用功能输出
- 通过配置寄存器，端口可配置成以下 8 种模式：

### GPIO 8 种工作模式

| 模式 | 性质 | 说明 |
|------|------|------|
| **浮空输入** | 数字输入 | 可读取引脚电平，若引脚悬空，则电平不确定 |
| **上拉输入** | 数字输入 | 内部连接上拉电阻，悬空时默认高电平 |
| **下拉输入** | 数字输入 | 内部连接下拉电阻，悬空时默认低电平 |
| **模拟输入** | 模拟输入 | GPIO 无效，引脚直接接入内部 ADC |
| **开漏输出** | 数字输出 | 可输出引脚电平，高电平为高阻态，低电平接 VSS |
| **推挽输出** | 数字输出 | 可输出引脚电平，高电平接 VDD，低电平接 VSS |
| **复用开漏输出** | 数字输出 | 由片上外设控制，高电平为高阻态，低电平接 VSS |
| **复用推挽输出** | 数字输出 | 由片上外设控制，高电平接 VDD，低电平接 VSS |

## 3-2 LED闪烁 & LED流水灯 & 蜂鸣器

### GPIO 常用库函数

```c
GPIO_SetBits();                    // 把指定端口设置为高电平
GPIO_ResetBits();                  // 把指定端口设置为低电平
GPIO_WriteBit(GPIOx, GPIO_Pin, BitVal);  // 根据第三个参数的值设置指定端口
GPIO_Write(GPIOx, PortVal);        // 同时对 16 个端口进行写入操作
```
