# 第14章 WDG看门狗

## 14-1 独立看门狗 IWDG

### WDG 简介

- WDG（Watchdog）看门狗，本质上是一个计数器
- 在指定时间范围内，程序没有执行喂狗（重置计数器）操作时，看门狗硬件电路就自动产生复位信号
- STM32 内置两个看门狗：
  - **独立看门狗（IWDG）**：独立工作，对时间精度要求较低
  - **窗口看门狗（WWDG）**：要求看门狗在精确计时窗口起作用

### IWDG 独立看门狗

- 独立工作，使用内部低速时钟 LSI（40kHz）
- 对时间精度要求较低，用于检测程序跑飞、死循环等
- 12 位递减计数器

### IWDG 超时时间
_IWDG = T_LSI × PR预分频系数 × (RL + 1)
其中：T_LSI = 1 / F_LSI = 1 / 40kHz = 0.025ms

- 预分频系数：4、8、32、64、128、256
- 计数器：12 位
- 超时时间范围：0.1ms ~ 26214.4ms
- 喂狗方式：写键寄存器，重装固定值 RLR
- 防误操作：键寄存器和写保护

### IWDG 库函数

```c
IWDG_WriteAccessCmd();     // 写使能控制（解除写保护）
IWDG_SetPrescaler();       // 写预分频器
IWDG_SetReload();          // 写重装值
IWDG_ReloadCounter();      // 重新装载寄存器（喂狗）
IWDG_Enable();             // 启用独立看门狗
```

## 14-2 窗口看门狗 WWDG
### WWDG 窗口看门狗
使用 PCLK1（36MHz）作为时钟源
6 位有效计数器（T[5:0]）
具有"窗口"限制：必须在指定时间窗口内喂狗，过早或过晚都会复位
支持早期唤醒中断（EWI）
### WWDG 超时时间与窗口时间
超时时间：T_WWDG = T_PCLK1 × 4096 × WDGTB预分频系数 × (T[5:0] + 1)

窗口时间：T_WIN = T_PCLK1 × 4096 × WDGTB预分频系数 × (T[5:0] - W[5:0])

其中：T_PCLK1 = 1 / F_PCLK1 = 1 / 36MHz
预分频系数：1、2、4、8
计数器：6 位（有效计数）
超时时间范围：113us ~ 58.25ms
喂狗方式：直接写入计数器，写多少重装多少
防误操作：无
### WWDG 库函数
```c
WWDG_DeInit();             // 恢复缺省配置
WWDG_SetPrescaler();       // 写入预分频器
WWDG_SetWindowValue();     // 写入窗口值
WWDG_EnableIT();           // 使能早期唤醒中断
WWDG_SetCounter();         // 写入计数器（喂狗）
WWDG_Enable();             // 使能窗口看门狗
```
### IWDG vs WWDG 对比
| 项目        | IWDG 独立看门狗        | WWDG 窗口看门狗                  |
| --------- | ----------------- | --------------------------- |
| **复位条件**  | 计数器减到 0 后         | 计数器 T\[5:0] 减到 0 后，或过早重装计数器 |
| **中断**    | 无                 | 早期唤醒中断                      |
| **时钟源**   | LSI（40kHz）        | PCLK1（36MHz）                |
| **预分频系数** | 4、8、32、64、128、256 | 1、2、4、8                     |
| **计数器**   | 12 位              | 6 位（有效计数）                   |
| **超时时间**  | 0.1ms ~ 26214.4ms | 113us ~ 58.25ms             |
| **喂狗方式**  | 写键寄存器，重装固定值 RLR   | 直接写入计数器，写多少重装多少             |
| **防误操作**  | 键寄存器和写保护          | 无                           |
| **用途**    | 独立工作，对时间精度要求较低    | 要求看门狗在精确时间窗口起作用             |
